Conversor Universal

Um aplicativo web moderno e responsivo para conversão entre diferentes unidades como temperatura, distância, peso, tempo e moeda, desenvolvido em React + Vite.
Este projeto tem o objetivo de demonstrar domínio de frontend moderno, componentização e CSS puro, sem dependências externas de UI.

Tecnologias utilizadas

React

Vite

CSS Puro

JavaScript ES6+

Funcionalidades

Conversão entre várias categorias:

Temperatura: Celsius, Fahrenheit, Kelvin

Distância: Quilômetros, metros, centímetros, milhas, pés

Peso: Quilogramas, gramas, libras, onças

Tempo: Dias, horas, minutos, segundos

Moeda: BRL, USD, EUR (taxas fixas demonstrativas)

Interface limpa e responsiva

Troca automática entre unidades (De / Para)

Resultados com arredondamento e formatação local (pt-BR)

Totalmente funcional sem backend ou banco de dados
